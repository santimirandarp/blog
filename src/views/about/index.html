This Website

After several days working together, we manage to get a very basic website up and running.

The website was done from scratch and with just 2$USD for the first year. The domain name was bought in GoDaddy.

The site is hosted on a EC2 instance on Amazon Cloud. The backend is a Node/Express server, linked to a MongoDB database to store comments. The server uses ejs template engine to render HTML on the fly.

The frontend style is written in Sass. It is not using frameworks but Bootstrap or Tailwind and JQuery may be implemented.

The source code can be found at Github.
