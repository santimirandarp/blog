This Website

We were curious about how to create a website from the ground up, and noew - after several months, it's here. The tools and basic steps we followed to build the site are:

  • Buy a domain name in Godaddy (for about a dollar a year.)
  • Set up an EC2 instance on Amazon Cloud (this is a computer that we access remotely, and the one serving the website). There is a configuration overhead for opening connection ports (80 and 43). But it is for free (for a year).
  • Backend for handling http(s) requests. We use Node/Express.
  • Database to store information and communicate with the backend. We use MongoDB-Atlas and Mongoose.
  • Because blogposts follow a similar layout, we use a template engine that greatly simplifies the structure of the blog. We used express-handlebars but later switched to ejs.
  • The frontend stylesheets are compiled to CSS from Sass.
  • On development Gulp 4 and Webpack are used to handle tedious tasks.

All the code we execute is open to anyone for reading. The source code can be found at Github.
