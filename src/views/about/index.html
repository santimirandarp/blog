This Website

After several days working together, we manage to get a very basic website up and running.

The website was done from scratch, the domain name was bought in GoDaddy.

The site is hosted on a EC2 instance on Amazon Cloud. The backend is a Node/Express server, linked to a MongoDB database. The server uses ejs template engine to render HTML on the fly.

The frontend stylesheets are compiled to CSS from Sass. Bootstrap, Tailwind, JQuery may be added in the future.

The source code can be found at Github.
